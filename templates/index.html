Ensemble Analysis Discussion

In this classification setup, our base models achieved complementary performance: Decision Tree ($F_1 = 95.20\%$, $AUC = 0.9826$), Logistic Regression ($F_1 = 92.84\%$, $AUC = 0.9775$), and Perceptron ($F_1 = 90.31\%$, $AUC = 0.9680$). Since the base models have complementary strengths (e.g. Perceptron learns a simple linear partition, Logistic Regression models smooth probabilities, and Decision Tree captures step thresholds), the Level-1 Stacking Meta-model successfully blended their OOF probabilities to achieve a superior performance of $F_1 = 96.11\%$ and an outstanding $AUC = 0.9938$.

This result clearly demonstrates the power of stacking. By combining models with different architectural biases, the meta-learner assigns coefficients that leverage each model's confidence scores, reducing ensemble variance and boosting classification generalization beyond any individual base model.

Scouting Calibration Rules
  • Elite Target Threshold: Predicts if a player is rated Overall $\ge 75$ using only skill attributes.
  • Zero-Leakage Guarantee: Preprocessing pipelines (imputers and scalers) were fit strictly on fold splits, preventing future test details from bleeding into training coefficients.
  • Soft Probabilities: Base classifier outputs are transferred as continuous probability scores ($[0,1]$) rather than binary integers, maintaining classification certainty.
Stacking Architecture & Out-Of-Fold Pipeline

To train the Level-1 meta-model without causing target leakage, we must generate predictions for the training set such that the base models predict on data they were never trained on. We achieve this manually using K-Fold Out-Of-Fold (OOF) CV.

1
Fold Splitting
The training dataset is split into 5 equal portions (folds). For each fold, 4 slices are used to train, and 1 slice is held out as the validation set.
2
Leak-Free Scaling
Crucially, we fit our scaler and imputer ONLY on the 4 training slices, and then apply it to the validation slice. This prevents statistical parameters from leaking.
CRITICAL STEP: Normal pre-scaling violates fold boundaries!
3
OOF Predictions
The base models predict on the validation slice. By looping through all 5 folds, we generate predictions for every training row. These form the input to train Level-1.
